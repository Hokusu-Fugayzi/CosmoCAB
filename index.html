Schemes

A scheme defines the catalog, construction method, and finish style used for cabinet orders.

Layouts

Room configuration templates. One layout can be shared by many units — create it once, assign it to as many apartments as needed.

💡 Layouts are reusable templates. For example, create one "Standard 2BR Kitchen" layout and assign it to all 80 two-bedroom units. You don't need a separate layout per apartment.

Units

Individual apartments or rooms. Each unit is assigned a layout template and tracked through production to delivery.
